# Блог на Django

Этот тестовый блог написан на:

- Django
- Docker
- Postgres

Для развертывания блога используется Docker и Postgres.

`docker-compose up`

Блог предназначен для демонстрации моих навыков в веб-разработке. Он имеет простой интерфейс и предоставляет пользователям возможность публиковать свои статьи, комментировать их и обмениваться мнениями.

Я продолжаю работать над улучшением блога, добавляя новые функции и улучшая производительность.
